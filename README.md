PHP Games
=======

Mini games in PHP terminal. Includes:

1. a **Calculator**
2. **True or False**
3. **Questions / Answers**
4. **Mastermind**


Language
=======
Mini games is only available in French.

Launch
======
To launch the program, download this repository, open a terminal, go in the root of the project and run:

```sh
	php index.php
```
